# PHAROS-AI Platform

**Plataforma de alfabetización crítica en IA para personas adultas**
Proyecto Erasmus+ · Consorcio UEC · INAPP · OCECAN

---

## Inicio rápido

```bash
# 1. Clona el repositorio y entra en la carpeta
git clone https://github.com/yorch2023/yorch2023.git
cd yorch2023

# 2. Ejecuta el setup
bash scripts/setup.sh

# 3. Edita .env con tus credenciales
nano .env

# 4. Inicia el middleware IA en modo desarrollo
npm run dev
```

El middleware IA estará disponible en `http://localhost:3001`.
